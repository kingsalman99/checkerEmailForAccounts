A script for checking the availability of an e-mail from vk.com, ok.ru, steampowered.com, ea.com

<h3>MAIL.RU is not supported</h3> 

<h3>Usage:</h3>
main.py -O path/to/emails.txt --vk --steam --ok --origin

<h3>Input format:</h3>
email@email.com:MEGAPASSWORD
